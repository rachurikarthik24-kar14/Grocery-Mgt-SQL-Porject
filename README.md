# SQL Recruitment Management System

# Project Overview

This project is a Recruitment Management System built using MySQL. It demonstrates the design and implementation of a relational database to manage recruitment-related data efficiently while applying SQL concepts to generate meaningful business insights.

# Objectives
• Design a relational database using MySQL.
• Build relationships using Primary Keys and Foreign Keys.
• Perform data analysis using SQL queries.
• Generate business insights from structured data.

# Technologies Used
• MySQL
• SQL

# Database Structure
The database consists of 7 interconnected tables with properly defined relationships to ensure data integrity and efficient data management.

# Key Features
• Designed a relational database using MySQL.
• Created 7 normalized tables.
• Implemented Primary Key and Foreign Key relationships.
• Maintained data integrity through normalization.
• Performed business-oriented data analysis.

# SQL Concepts Used
• INNER JOIN.
• LEFT JOIN.
• RIGHT JOIN.
• Aggregate Functions (COUNT, SUM, AVG, MIN, MAX).
• GROUP BY.
• HAVING.
• WHERE.
• ORDER BY.
• Subqueries.

# Business Insights

• Evaluated product performance.
• Analyzed supplier contributions.
• Measured employee productivity.
• Analyzed sales trends for better decision-making.

# Key Learning Outcomes
• Improved relational database design skills.
• Strengthened SQL querying techniques.
• Gained hands-on experience with real-world business scenarios.
• Enhanced analytical and problem-solving abilities.
